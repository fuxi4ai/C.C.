## 三、框架层陷阱

### G-06：虚假独立的多框架收敛
**描述**：援引多个"独立"框架指向同一结论，被当作增强可信度的证据。但如果这些框架共享同一个底层假设，它们实际上是同一个论证的不同语言表述，不是真正独立的证据线。

**案例**：《棋局之外》的四个框架（Cui Bono、马汉、麦金德、二战教训）都预设"美国行为以长期结构性利益为导向"，这个底层假设未被独立检验，却被当作四重证据。

**警报信号**：当多个框架从不互相冲突时，考虑它们是否真的独立。真正独立的框架在某些情境下应该给出不同甚至对立的预测。

**修复方向**：明确列出每个框架的底层假设，检查假设集合的重叠度。

---

### G-07：目标多元与执行统一化的逻辑矛盾
**描述**：论证一方面说"多个利益集团各自追求局部利益，没有总设计师"，另一方面又援引执行层的高度统一化（清洗将领、改名战争部、万亿军费）作为证据，两个命题之间存在张力，未被正面处理。

**修复方向**：明确区分"目标层"（多元，无协调）和"执行层"（可能统一化），并解释两者如何共存，而非混用。

## 四、方法论陷阱

### G-08：受益→意图的未论证跳跃
**描述**：从"谁从某事件中获益"直接推断"谁意图使该事件发生"，这个逻辑跳跃需要独立论证，不能作为默认步骤。

**原则**：受益是意图的必要条件，但不是充分条件。历史上大量的"受益者"是机会主义者，而非设计者。

**测试问题**：如果没有任何主观意图，纯粹的结构性力量、历史惯性、信息不足，能否产生完全相同的观察结果？如果答案是"能"，则意图假设缺乏独立支撑。

---

### G-09：奥卡姆剃刀被选择性使用
**描述**：分析中援引奥卡姆剃刀原则批评替代假说过于简单，但自身引入的"结构性偏好"概念实际上比"制度惯性+国际结构压力"更复杂，而解释力并未显著更高。

**修复方向**：在每次引入新的解释性概念时，明确说明它相较于更简单的替代解释，额外解释了哪些现象。

---

## 五、元层面陷阱

### G-10：框架自我保护（反身性盲点）
**描述**：当一个框架被广泛使用并获得足够知名度后，分析对象（政策行为者）会开始将"被分析"纳入自己的策略。框架因此可能在解释过去的同时，改变了未来的现象。这是社会科学中最深的循环。

**意义**：这不是放弃分析的理由，而是提醒分析者：框架的解释力有时效性，需要定期检验其是否仍然匹配现实。

---

### G-11：分析单元层次混淆（族群/国家/机构）
**描述**：将特定行为主体（国家政府、具体机构、特定个人）的行为归因于其所属的更大群体（民族、宗教、文化群体），是G-08（受益→意图跳跃）在群体层面的特殊表现。

**四层标准**：分析单元必须与证据层次匹配。
- **Layer 1** 国家行为体（政府/军队/外交/立法）→ 充分可分析
- **Layer 2** 特定组织（AIPAC、特定党派、智库、教会等）→ 充分可分析
- **Layer 3** 具体个人/机构（具体政治人物、具体公司、具体金融机构）→ 按标准分析
- **Layer 4** 族群/宗教/文明整体（"犹太人"、"美国人"、"穆斯林"、"中国人"整体意图）→ **允许输出，但承担更高证据责任**（见下方规则修订）

**Layer 4 的修订规则（2026-05-09 用户裁定）**

原规则"Layer 4 分析失效/禁止"过于严格——它把一种**合法的碰撞对象**排除在项目之外。按项目根本哲学（"思维的碰撞本身即意义"），Layer 4 应作为可讨论对象保留，但**必须承担诚实成本**：

- **强制标注**：任何涉及族群/宗教/文明整体的陈述必须**显式**标注 `[Layer 4]`
- **强制证据责任**：陈述必须附：(a) 该陈述所依据的具体行为主体证据（来自 Layer 1/2/3）；(b) 该陈述的可证伪条件；(c) 该群体内部已知的显著异质性（反例方向）
- **禁止未标注的集体主语**："犹太人做了 X"、"穆斯林谋求 Y"、"中国人认为 Z" 未附 `[Layer 4]` 标签 → 依然是违规
- **禁止把 Layer 4 当作因果解释的终点**：Layer 4 陈述可以作为**观察性假说**或**描述性概括**，不能直接作为机制推论的基础——Layer 4 → 具体行为归因的链条必须经过 Layer 1/2/3 的中介

**标注格式示例**：
```
[Layer 4] 在现代史上，散居犹太社群普遍对反犹信号保持高度敏感。
依据：[Layer 2] Anti-Defamation League 年度统计显示 ...；
     [Layer 3] 具体个人回忆录（Elie Wiesel、Philip Roth 等）。
可证伪条件：若某犹太社群在反犹信号明确时不激活防御反应，则削弱该 Layer 4 描述。
异质性：正统派与改革派、美国犹太人与以色列犹太人、Ashkenazi 与 Mizrahi 在此问题上有显著分化。
```

**适用范围**：此规则适用于所有族群/宗教/文明类 Layer 4 陈述——犹太人、穆斯林、中国人、美国人、白人、盎格鲁-撒克逊等。

**警报信号**：分析中出现"XX民族/宗教群体在谋求..."这类表述时，立即追问：
1. 是否附了 `[Layer 4]` 标签？
2. 是否列出了 Layer 1/2/3 的依据？
3. 是否附了可证伪条件与内部异质性？
缺任何一项，返工。

### G-12：精密度抬升陷阱（Precision Elevation Fallacy）

**描述**：当分析者遇到**特别精密、密集、设计化**的现象时，倾向于将其标记为"新维度 / 新层级 / 新机制 / 需要修订框架的根本性发现"，而实际上精密度是**术层表现**，不构成本层新机制。这是 G-03（对话上调偏误）的技术化变形——不由证据累积驱动，而由**细节密度的美学冲击**驱动。

**核心诊断**：当你说"这个现象颠覆了框架的隐含假设"时，请先追问——**颠覆的是术层表现，还是本层机制？** 如果现象只是让术层表现更精密，那么框架没有被颠覆，只是被一次术层高密度运作具象化了。

**典型案例（2026-05-12 首次触发）**：

CS-08 "大日本帝国"命名分析（v1.0）中，Claude 对该命名的六层叠加符号学、58 年完整生命周期、制度设计型特征印象深刻，提议：
- F-07 框架新增 L3.8 "设计化自称"条款
- a.7 新增第七子形态"施行性命名"
- F-07 原隐含假设"文明身份是历史累积的"需根本性修订

**Doctor 2026-05-12 纠偏**：
> "工程是历史的一部分，是突出的历史事件，依然归于术。其质心仍是民族基因、推动者身份/利益/需求。"

**按此纠偏的质心分析**：
- 六层叠加 → 日本基因库（天皇神圣 + 汉字载体 + 实用主义综合 + 国学 + 双重正当性）的既有项召唤，没有一层是凭空设计
- 命名工程 → 萨长藩阀 + 伊藤博文/山县有朋/井上毅/明治天皇的具体身份利益需求的符号化表达
- 民众动员 → L1.补充 第二条动态反馈的具体运作
- **结论**：既有 L1.补充 第一至第四条元机制完全覆盖此现象，无需任何新维度

**警报信号**：
1. "这个现象的精密程度前所未有"——警报（精密度本身不是理论层级标志）
2. "需要新立 L3.X / 第 N 条元机制 / 新子形态"——警报（检查是否可归入既有术层子条款）
3. "这触发 F-XX 框架的根本性修订"——高度警报（检查本层机制是否真的改变，还是仅术层表现变了）
4. "这个案例太独特，不能用既有分类"——警报（真正独特的是本层机制还是术层形态？）

**对抗机制（强制清单）**：
遇到被标记为"可能是新维度"的精密现象时，在提议立案前必须追问：
- [ ] 本层质心是什么？（基因库 + 推动者身份利益需求 + 动态反馈三层分别拆解）
- [ ] 既有元机制（L1.补充 第一至第五条）是否完全覆盖此质心？
- [ ] 精密度是**本层新机制的证据**，还是**既有机制在特定条件下的高密度运作**？
- [ ] 如果归为术层子说明能否同样解释此现象？能则必须归为术层。

**修复方向**：
- 新立案候选提出后，**必须完成术/本二分的显式拆解**——术层表现 + 本层质心分开列出
- 若本层质心能被既有机制完全覆盖，**必须降级为既有机制的术层子说明**，不新增层级
- 形式化要求：任何新立案提议的起始段必须有"本层新增了什么"的明确回答，否则自动归入术层

**与既有陷阱的关系**：
- 与 G-03（对话上调偏误）：G-03 由论据累积驱动上调，G-12 由细节密度驱动；两者都需对抗性提问
- 与 G-01（万能概念）：G-12 是**反向**——不是概念太宽能解释一切，而是现象太精密被误为新概念
- 与 G-10（框架自我保护）：G-10 抗拒修订，G-12 过度热衷修订；两者都是认识论失衡

**入档来源**：Doctor 2026-05-12 对"大日本帝国跨文明独特性"分析的纠偏对话 · 首例入档见 CS-08/共享基础/历史时间线.md §XV.6

### G-13：部分纠偏误为完整纠偏陷阱（Partial-for-Whole Correction Fallacy）

**描述**：分析者在评估某文明/制度的自我纠偏能力时，倾向于把**某一维度的强纠偏**（如基建 / 反腐 / 危机响应 / 经济增长）推论为**全维度强纠偏**，忽略其他维度的系统性弱表现。这是 G-08（受益→意图跳跃）在制度评估层面的特殊变形——不是从受益跳跃到意图，而是从部分维度跳跃到整体评价。

**核心诊断**：当你对某制度案例做整体判断时，请先追问——**我是否已对第六条七项维度分别评估？是否存在某维度的强表现被我当作整体代表？** 强纠偏 ≠ 零代价纠偏；强维度 ≠ 全维度强。

**典型案例（2026-05-12 首次触发）**：

Doctor 中华视角对中国土地国有化的主动暴露揭示了 G-13：
- **强维度**：基建能力（高铁网 4.5 万公里）/ 规划能力（浦东 / 深圳 / 雄安）/ 危机响应（2020 疫情方舱 10 天建成）/ 脱贫攻坚（960 万人易地搬迁）
- **弱维度**：纠偏代价分布（第 ⑤ 维度）——三峡移民 127 万、强拆历史、少数群体权益压制代价
- **错误路径**：用强维度的成功（整体经济发展 + 基建成就）掩盖弱维度的系统性代价
- **Doctor 纠正**：即便有经济补偿，土地国有化在执行层面仍牺牲了少数人权益——这是真实代价，不能被"大多数人受益"叙事消解

**警报信号**：
- 使用"整体成功 / 模式优越 / 历史传承"等整体性叙事评价某制度
- 跨维度评估缺席——只讨论基建不讨论言论 · 只讨论效率不讨论公平
- 批评时说"虽然有 X 问题，但整体上 Y" ——这一句式往往掩盖 X 的系统性
- 用其他制度的问题反推自身制度的正当性（"美国也这样" / "中国也这样"）

**对抗机制**：
- **强制七维度分别评估** · 每项维度单独诊断 + 单独打分
- **主动暴露弱维度** · 分析者须识别自身视角下容易辩护的维度 + 主动列出其代价
- **代价分布核查** · 强维度的成功是否伴随弱维度的代价集中
- **跨视角交叉** · 从对立视角重新审视自己的整体判断

**与既有陷阱的关系**：
- 与 G-05（成本缺席）：G-05 是个案分析的成本忽略；G-13 是制度评估层面的维度忽略
- 与 G-08（受益→意图跳跃）：G-08 是因果链跳跃；G-13 是评估维度跳跃——两者都是**从局部推全局**
- 与 G-12（精密度抬升）/ G-14（程序完备误为实质）：G-12/G-13/G-14 共同构成"形式→实质陷阱三联"——G-12 是精密度诱惑，G-13 是部分维度代表，G-14 是程序代表实质

**入档来源**：Doctor 2026-05-12 第三次精化（土地国有化代价）· 第四次精化（大家长制条件性成功 + 跨代际风险）对话

### G-14：程序完备误为实质成功陷阱（Procedural-for-Substantive Success Fallacy）

**描述**：分析者将制度**程序的复杂性、审慎性、多方参与、长审查期**当作**实质判断成功**的证据——但程序完备可能掩盖实质判断失败。程序运作**可以完备**却**不匹配实质比例性**（阻止的行为规模 vs 真实伤害 vs 替代方案空间）。

**核心诊断**：当你判定某制度案例"成功"时，请先追问——**它达到了制度设计的实质目的吗？还是仅达到了程序自身的延续？** 程序完备是必要条件，不是充分条件。

**典型案例（2026-05-12 首次触发）**：

美国五大湖水资源治理（Claude v1.0 对第六条候选的先前判断）：
- **程序表现**：2008 Great Lakes Compact 国际条约 · 8 州一票否决制 · 多方环保组织参与 · 长期审查机制
- **Claude 先前判断**：把这一"程序完备"误为"少数保护取向的成功案例"
- **Doctor 水文质疑**：2010 Waukesha 调水案——申请量 **0.014 立方公里/年 = 五大湖年流出 212 立方公里的 0.0066%** · 审查 **6 年** · 五大湖总蓄水量 22,684 立方公里（自然波动远大于人为抽取）
- **实质判断**：对 0.0066% 规模的调水申请做 6 年审查 · 同时 Ogallala 含水层（美国地下水 30%）因无统一监管而不可逆衰竭——**同一分散制度的双重失败**
- **修订后定位**：五大湖不是"少数保护成功"，而是"过度保护端 + 缺乏比例性判断"的病态；与 Ogallala 的"保护不足端"形成对称病态

**警报信号**：
- 用"多方同意 / 长期审查 / 广泛参与 / 合规"描述制度成功
- 不计算实质比例（申请规模 / 总量 / 边际成本）
- 不比较"被阻止的"与"已发生的"真实后果
- 混淆"制度运作正常"与"制度达成实质目的"

**对抗机制**：
- **强制计算实质比例**：规模 / 边际成本 / 机会成本量化
- **对比实际后果**：被阻止的行为 vs 未被阻止的行为的真实伤害对称校验
- **追问程序目的**：这个程序达到其设计目的了吗？还是只达到了程序自身的延续？
- **替代方案空间**：是否允许讨论中间方案（如限量 + 监测）而非非此即彼

**与既有陷阱的关系**：
- 与 G-12（精密度抬升）：G-12 把术层精密度误为本层新机制；G-14 把程序完备误为实质判断成功——**不同层级的形式→实质错误**
- 与 G-13（部分纠偏误为完整纠偏）：G-13 是维度层级的部分→整体；G-14 是形式→实质层级的部分→整体
- 与 G-10（框架自我保护）：G-14 是**反向**——不是抗拒修订，而是把程序稳定当作实质成功
