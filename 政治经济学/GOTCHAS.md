## 分组导航

### 一、概念与方法论基础（G-01~G-05）

**所在文件**：`Projects/政治经济学/GOTCHAS.md` §一~§二（已收入通用教训）

- **G-01 万能概念陷阱** — 防御工具：要求每个概念附明确反例条件
- **G-02 主动/被动混用** — 防御工具：每条因果命题标注证据标准
- **G-03 对话动力上调偏误** — 防御工具：定期召唤质疑者子 agent + 单向上调警报
- **G-04 事后拟合** — 防御工具：`predictions-register.md` 事前可证伪预测登记
- **G-05 成本缺席** — 防御工具：受益分析必须配对成本端核查

### 二、框架层陷阱（G-06~G-07）

**所在文件**：[[陷阱-G-06-G-14]] §三

- **G-06 虚假独立的多框架收敛** — 检查：每个框架的底层假设是否真不相交
- **G-07 目标多元 vs 执行统一化的矛盾** — 检查：明确区分"目标层"与"执行层"

### 三、方法论陷阱（G-08~G-09）

**所在文件**：[[陷阱-G-06-G-14]] §四

- **G-08 受益→意图未论证跳跃** — 测试问题："无主观意图能否产生相同结果？"
- **G-09 奥卡姆剃刀选择性使用** — 检查：新概念额外解释了哪些现象？

### 四、元层面陷阱（G-10~G-12）

**所在文件**：[[陷阱-G-06-G-14]] §五

- **G-10 框架自我保护** — 提醒：框架解释力有时效性，需定期检验
- **G-11 分析单元层次混淆** — **规则修订（2026-05-09 用户裁定）**：
  - Layer 1 国家行为体 → 充分可分析
  - Layer 2 特定组织（AIPAC 等） → 充分可分析
  - Layer 3 具体个人/机构 → 按标准分析
  - **Layer 4 族群/宗教/文明整体 → 允许输出，但必须 `[Layer 4]` 显式标注 + 列 Layer 1/2/3 依据 + 可证伪条件 + 内部异质性**
- **G-12 精密度抬升陷阱**（2026-05-12 立案） — 强制对抗清单：本层质心是什么？既有元机制是否覆盖？精密度是新机制还是高密度运作？

### 五、制度评估陷阱（G-13~G-14）—— "形式→实质陷阱三联"

**所在文件**：[[陷阱-G-06-G-14]] §五（G-13/G-14 与 G-12 并列）

- **G-13 部分纠偏误为完整纠偏**（2026-05-12 立案） — 强制：七维度分别评估 + 主动暴露弱维度
- **G-14 程序完备误为实质成功**（2026-05-12 立案） — 强制：计算实质比例（规模/边际/机会成本）+ 比较实际后果

**三联关系**：
- G-12 = 精密度诱惑（术层→本层错误）
- G-13 = 部分代表整体（维度层级）
- G-14 = 程序代表实质（形式→实质层级）

> **Claude 训练数据特别警戒**（CS-08 §六 方法论警戒第 11 项）：英语训练数据对西方制度的程序完备有结构性加权——环保主义叙事 / 多方参与叙事 / 审慎审查叙事被系统性合理化。对西方制度案例须**强制引入比例性诊断**。
