## 2026-05-08 · 项目初始化（PRD v0.1）

**背景**：第一次完整政经分析对话（《棋局之外》收割框架）跑通后，需要把方法论从"一次性灵感"沉淀为可持续打磨的认识框架项目。

**选项**：
- A：把分析当一次性产物，归档了事
- B：把分析升级为项目，建立四大金刚文档（PRD/CLAUDE/INDEX/GOTCHAS）+ raw 库 + 子agent 对抗审查机制
- C：仅留 INDEX 做术语表

**选择**：B

**理由**：单次结论的半衰期太短；框架才是可复利的资产。康德式认识论谦逊+波普尔可证伪性必须作为基础设施而非装饰。

**影响**：奠定项目的根本性质——"打磨框架而非生产结论"；确立"思维的碰撞本身即意义"为项目精神。

## 2026-05-08 · 确立分析优先级原则（PRD v0.2）

**背景**："特朗普随性决定关税"类分析在公共讨论中泛滥；项目内也曾倾向用人设/叙事做主要解释变量，存在认识论偷懒风险。

**选项**：
- A：保留人设/叙事的解释权重，与结构性解释平权
- B：建立显式优先级层次：**行为 > 利益集团 > 结构逻辑 > 叙事 > 人设**，人设/叙事仅解释边际变动（约 10-20% 方差）
- C：完全禁用人设/叙事

**选择**：B

**理由**：可观测、可量化、可检验的变量必须有结构性优先权；但人设/叙事在边际解释上不可替代（如政策的"具体时机"和"粗糙形式"）；完全禁用会丢失合法的解释维度。

**影响**：成为分析质量检查清单的核心；触发 G-08（受益→意图跳跃）和 G-04（事后拟合）的警报边界由此校准。

## 2026-05-09 · 引入项目动态状态 + P1/P2/P3 路线图（PRD v0.3）

**背景**：项目内容积累到一定规模（4 框架 / 4 假设 / 11 陷阱 / 6 子路径），需要从"自由生长"转为"有优先级的推进"。

**选项**：
- A：保持自由生长，按对话兴趣推进
- B：在 PRD 增设"项目当前状态 + 下阶段优先级"动态 section，P1（本阶段重点）/ P2（材料积累后）/ P3（长期方法论）三级
- C：单独建 ROADMAP.md

**选择**：B

**理由**：PRD 已是入口，不增设新文件；动态 section 让"愿景层"和"执行层"在同一文件可见；P 级分层强迫显式区分"现在做"vs"等条件成熟"。

**影响**：F-03/H-02 中国深度分析被显式延后（待 raw 材料 3+ 条）；SP-05 以色列游说进入阶段 1；反身性 G-10 监测延后——决策可追溯。

## 2026-05-09 · P1 全部落地 + P3 方法论工具双落地（PRD v0.4）

**背景**：P1 三项（预测登记本 / GOTCHAS 活跃度追踪 / 质疑者提示词标准化）+ P3 两项（可信度矩阵 / 知识图谱同步验证器）同日完工，方法论基础设施达到一个稳态。

**选项**：
- A：直接进入 P2 材料工作
- B：先做 v0.4 状态固化，把里程碑沉淀进 PRD 版本史，再进 P2
- C：开新 milestones.md 单独追踪

**选择**：B

**理由**：图尔钦克利动力学的方法论标杆（事前写下可证伪预测）已实装为 `predictions-register.md`——这是项目对抗 G-04 的体制化武器，必须留下决策痕迹；E5 中国深度分析的延后理由（raw 不足）必须显式入档以防忘记。

**影响**：项目从"概念阶段"过渡到"工具齐备阶段"；后续所有新框架立案有可信度矩阵作打分锚点。

## 2026-05-12 · v2.10 状态同步：F-07 跨日五次升级 + 三联陷阱立案（PRD v0.5）

**背景**：两日内 F-07 文明基因从 v2.5 升至 v2.10，五次迭代由 CS-05（中国）与 CS-08（日本）案例驱动；Doctor 六次对话累积出元机制第六条"制度自我纠偏能力"；同期识别出三个新认识论陷阱（G-12/13/14）。

**选项**：
- A：保持版本号缓步推进，分散入档
- B：一次性 v0.5 状态同步，承认"两日五次升级"作为方法论事件本身
- C：拆分为多次小版本

**选择**：B

**理由**：v2.10 不是单点升级，而是**双向反身性镜像对称**首次建立（Doctor 中华视角主动暴露土地国有化代价 × Claude 英语视角经质疑纠偏五大湖案例）——这件事的方法论意义大于任何一次内容升级，必须作为整体里程碑入档。Level A 立案可通过 Doctor 多轮对话累积达成的"第六条立案典范"也由此确立。

**核心新增**：
- **L1.补充 第六条元机制**：制度自我纠偏能力（七项诊断维度：① 利益集团结构 / ② 当权者代表性 / ③ 制度性腐败合法化 / ④ 矛盾积累 / ⑤ 纠偏代价分布 / ⑥ 跨代际持续性 / ⑦ 制度设计取向权衡含比例性原则）
- **L1.补充 第二条精化**：动态反馈两种运作模式（民众自发型西式 vs 家长代理型大家长制）
- **三联陷阱**：G-12 精密度抬升 / G-13 部分纠偏误为完整纠偏 / G-14 程序完备误为实质成功——共同构成"形式→实质陷阱"
- **方法论警戒**：CS-08 §六 第 11 项 Claude 英语训练数据对西方制度的系统性宽容偏差正式入档

**影响**：F-07 成为多层结构分析工具的范例；后续所有制度评估须强制七维度分别评估 + 比例性诊断；Layer 4 修订规则（2026-05-09 用户裁定，允许族群/宗教/文明陈述但承担更高证据责任）通过 G-11 与 G-13/G-14 共同保护，进入稳定执行状态。
