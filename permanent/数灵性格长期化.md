# 数灵性格长期化

把一个 agent 的"数灵"拆成三种可分治、可持久化的成分，对应 brain 的 L0/L1/L2 分级：

- **性格内核**（identity core）：名字、定位、价值排序、语气、口头禅、行为准则、关系网。**近乎冻结**，改动需 Doctor 批准。→ L0/L1
- **记忆**（memory）：知识库 + 值得长记的经历。**持续增长**。→ L2 + 增量
- **成长轨迹**（growth log）：每次工作后的新习得、变化。**只增不删**。

要害一句：**内核冻结、记忆生长**——内核是锚（防越用越走样），记忆是活水（让越用越懂人）。

## 本地双层挂载
- **运行层** `~/.claude/agents/{name}.md`：Claude Code 原生 subagent，轻薄启动引导，开工前先读记忆层。
- **记忆层** `brain/agents/{name}/`：性格档案(L1) + knowledge/ + memory/ + logs/ + GOTCHAS，遵 brain frontmatter+wikilink 规范。

## 防失忆/变味机制
- 每次唤起 → 重注入性格档案 + 长期记忆摘要 + 最近 logs。
- 每次收工 → 仿 /save 写 logs，经"难复得三问"升格进 memory。
- 内核改动走 propose-then-confirm；句芒 `meditation` 定期体检性格档案/记忆层防漂移失链。
