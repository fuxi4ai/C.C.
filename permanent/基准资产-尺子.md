# 基准资产 · 尺子

> 通用原理：把一条赛道里**流动性最强的共识龙头**当作该赛道的**机会成本基准 / hurdle**。工作样本 = [[AI基建成长股估值-斜率与顶部可见性|中际旭创（AI 算力/光互联）]]。

## 定义

**尺子 = 一条赛道的必要收益率下限。** 任何候选标的，若其投资价值 / 赔率 / 潜在收益不优于这把尺子，就无独立投资价值——**不如直接买尺子**。

## 成为合格尺子的四个条件（缺一不可）

1. **零信息差 = 充分定价**：共识名、覆盖充分、有效定价 → 其价即市场最优估计，拿它当参照才"公平"，不是在跟一个错价的东西比。
2. **高流动 · 无控盘**：能真实建仓的**可投**基准，非理论标的。量不进去的东西当不了尺子。
3. **纯 beta · 赛道最干净的表达**：拿别人跟它比 = 在问"你除了赛道 beta，还多给了什么"。
4. （隐含）**代表性**：是该景气细分最纯正的载体，而非边角料。

## 三条使用说明（免得量歪）

### ① 一把尺子只量一个维度
尺子只对"同赛道或相邻"的候选有效。拿不同风险因子的标的（如国产替代半导体、另一条周期）比赔率 = 拿温度计量长度，赔率不在同一根轴上。**效力限定在它自己的因子暴露内。**

### ② 尺子会热胀冷缩（刻度随周期变）
正因是充分定价的共识 beta，其**前瞻赔率随周期变**：景气甜蜜区里是把"高标准"尺子（潜在收益高、难被超越）；越靠近供需再平衡 / 顶部可见，自身前瞻赔率被压缩，**门槛自动降低**，更多替代标的"够得着"。
> **关键等式**：这把尺子的刻度 = [[AI基建成长股估值-斜率与顶部可见性|「斜率 × 顶部可见性」]]的赔率读数本身。尺子与估值框架是**同一个对象**。

### ③ 它是"占优筛"，不是"避风港"
候选要真"比尺子值得买"，只能靠尺子结构上给不了的东西：
- (a) **信息差**：覆盖度低、你有 edge 的标的；
- (b) **同一 beta 更高的赔率**：更纯 / 更早周期的环节；
- (c) **不同 beta**：分散价值。

若只是"尺子但更小、流动性更差、同一个故事" → 被**严格占优**，直接买尺子。
> **反面陷阱**：无控盘 / 高流动 / 满仓共识 = 它也是**最拥挤的默认仓**，赛道 de-rate 时人群从这儿夺门而出，回撤 beta 最大。**它量得准上行赔率，却挡不住供需再平衡的回撤**——买它当默认，是"流动性上安全、波动上不安全"的默认。

## 一句话

尺子量的是"同一条赛道里，你有没有比充分定价的共识 beta 更好的赔率"；刻度随周期伸缩、等于两轴赔率读数；是**占优筛**，不是避风港。
