# 核学说分层读法 + 判别器

> 原子概念 · 源自 2026-06-02 PEC GE-05「阻止日本拥核」第五轮（Doctor「模糊来袭即核报复，对有核国家，这是中国现行政策，可以检阅中文媒体」→ CC web 核中文源）。
> 用途：任何"某国核政策是 X"的命题，先分层，再按判别器定责。**核学说"声明 ≠ 操作"是普遍律**，对宣示性政策（如 NFU）尤甚。

## 三层（+ 一个旁层）

| 层 | 是什么 | 证据源 | 性质 |
|---|---|---|---|
| **声明层** | 官方对外宣示的政策 | 外交部文件 / 国防白皮书 / 条约提案 | 含**对外品牌/外交功能**，未必等于实际意图 |
| **操作/学理层** | 实际部署姿态 + 本国战略学界对其运作方式的描述 | 部队状态（弹头是否合一存放 / LOW vs ride-out）/ 权威学者（如李彬） | 比声明更接近实际，但仍受可观测性限制 |
| **战时姿态层** | 存续/危机场景下的实际意图 | 推演 + 姿态漂移证据（预警体系建设 / 库扩张 / 措辞松动）+ 场景自洽性 | 最难观测、最可能与声明背离 |
| **（旁）民族主义军评层** | 评论员/智库的强势主张 | 昆仑策 / 环球 / 知乎等 | **常是"改革倡议"——其"呼吁改"恰证它非现行官方政策**；不可抬成"现行政策" |

## 判别器

> **外交部 / 白皮书措辞　vs　民族主义军事评论。** 二者冲突时，按层归责，不混。

- 声明层与命题相反 → 命题在声明层**证伪**；
- 但官方**永不会声明**违背其外交品牌的政策（如违 NFU 的"模糊即核报复"），故**官方无声明对战时实际意图是弱证据**——不能据此在操作/战时层也判否。

## α / α′ 双向纪律（核心）

- **α′（防中文源默认）**：**禁止把单层民族主义军评抬成"现行政策"。** （活体：拿昆仑策"以核止战先手为强"当中国现行核政策。）
- **α（防西方默认）**：**也不拿旧的 categorical 宣示（如中国 1964 NFU + ride-out）当铁律**否定姿态的实质漂移。两端都拒绝。

## worked sample（中国"模糊来袭即核报复·对有核国家"）

- **声明层证伪**：2024-07 互不首先使用工作文件 + 2025-11 军控裁军白皮书＝无条件 categorical NFU + 推互不首先使用条约；李彬《中国核战略辨析》：中国传统**不搞 LOW**、ride-out 后数日才反击。
- **操作/战时层成立**：核学说"声明≠操作"普遍律 + 中国永不会声明此政策（砸 NFU 招牌）+ 预警体系/库扩张使姿态漂移 + 存续场景（对手已拥核）下转 LOW 高度可信；且若中国常规首击在先，对手回击是对华来袭，按"可能核打击"行核反击**在 NFU 措辞内自洽**（未"首先"用核）。
- **军评层**：昆仑策/知乎"要不要放弃 NFU"＝上升中的**改革倡议**，非现行官方政策。
- **净判**：声明层证伪 / 操作·战时层成立 / 军评层是倡议。**待补**：若有把它写成"政策"的官方或半官方源（非评论员）→ 上修。
