## 分析框架（严格按顺序）

```
政策与监管（5分）
→ 技术变革与供需（35分）
→ 竞争格局与产业链（25分）
→ 新赛道与未来预期（15分）
→ 估值与安全边际（15分）
→ 财务健康与风险排查（5分）
──────────────────────
总分：100分
```

## 使用规则（强制）

1. **必须先运行引擎**：`python five_forces_engine_v3.py <ts_code>`
2. 引擎完成 Tushare 拉取 + 渊图查询 + 财务自动打分
3. **禁止**绕过引擎直接调用 `get_tushare_data.reuse_conn.py`（已废弃）

- 调用[[渊图]]的语义检索接口
