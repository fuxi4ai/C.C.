# Case Study · 2026-07-21 A股「史诗级修复」资金面反演

> **一句话主结论**：Doctor 命题「国家队主导史诗级修复」**方向坐实、机制需纠偏**。7/16–7/21 是一次**六大方阵组合拳**（国家队ETF＋央企增持＋公募私募自购＋险资＋上市公司回购＋政策喊话），不是单点救市；**真正的量级来自宽基ETF的沉默买入（署名指向汇金系、无公告），而非那两家发了公告的央企（国新＋诚通合计约600亿）**。7/21 深V 的**直接扳机是半导体beta＋空头回补＋外围企稳**，国家队在修复日反而缩手——它的贡献在 7/16–7/20 **托住权重、防止指数踩踏**，而非择时拉起 7/21。政策面到 7/21 仍只满足「喊话」，未出「真金白银＋制度」，故这是一个 **「资金底」而非「政策底」**。

## 二 · 国家队直接踪迹：梯队分工，非轮值

**公开层面无制度化「轮值」，实际是梯队分工——各管一段、同时在场。**

### 2.1 明牌·央企系（发公告，声量大，金额有限）【A·官方】

- **国新**（国新投资）：7/19 通稿原文「已经使用股票回购增持专项再贷款及配套资金**超500亿元**，用于维护市场稳定…同时配以自有资金继续增持**中央企业股票**」。→ 存量口径 超500亿，标的=央企股票（未列ETF）。
- **诚通**（诚通资本/诚旸投资）：「聚焦国资央企**大额增持中国股票资产，累计买入近百亿元**…继续大额增持**国资央企和科技企业股票及ETF**」。→ 近百亿，含ETF。
- 合计口径三种并存：公告口径（超500亿+近百亿）／中新网「约600亿」／中国基金报「累计超500亿」。来源：新华网 20260719。

### 2.2 暗牌·汇金系（不公告，主力量级）【B·财新/Bloomberg】

## 五 · 反证与纠偏（不选边）

1. **「国家队拉起了7/21」是叙事、非因果**。7/21深V直接扳机＝半导体（10:10科创50翻红）＋隔夜费半企稳＋KOSPI/日经反弹＋空头回补；国家队修复日**反而缩手**（ETF成交环比降）。国家队作用在 **7/16–7/20托权重防踩踏**，7/21弹性由科技beta和融资盘回补贡献。**把7/21归功国家队会高估其择时能力。**
2. **课件（小鲍）持相反解读**，并列存档：认为是**市场自身快速深跌出清融资盘形成「市场底」**，国新诚通通稿「无金额、属喊话」，按「政策底三要素」（真金白银＋制度改变＋强力监管）尚不满足。**真相大概率叠加**：国家队托住系统性尾部，出清由杠杆自我了断完成。
3. **「连锁反应」一环查无实据**：2026年7月的**雪球敲入/DMA爆雷全网无可信报道**（仅2024旧案）。本 case **降级为「未证实」**，不入机制主链。
4. **数字纠偏**：创业板+7.05%＝自库单日第5大（非史上第一）；KOSPI+3.56%（非+4.5%）；摩根大通33亿美元查无出处；「史诗级/十年一遇」原词未见于主流媒体（媒体用「大奇迹日」，「史诗级」仅指ETF净流入）。

## 七 · 可观测信号清单（未来复用·平准入场识别）

供烛照九阴日报与未来同类事件反演复用。**平准资金入场的可观测代理**：

1. **宽基ETF单日净流入异常**（≥500亿/日 或 单周破纪录）——最直接，数据宝/财联社次日可得。
2. **指数—广度背离**：主要指数收红/微跌，而跌停或跌≥9%家数处于高位（如7/20沪指+0.85%×643家跌≥9%）——「托权重、弃个股」是平准资金的盘面指纹。
3. **沪深300ETF/科创50ETF逆势成交放大＋收红**，而同期个股普跌。
4. **央企增持公告集中披露**（周末新华社渠道）——可见声部，滞后于ETF流水。
5. **权重股尾盘异动拉升**（护盘时点特征）。
6. **期指基差**：护盘期现货多头＋期指空头组合会压中小盘期指贴水走阔（IM/IC＜IF/IH）。

⚠️ **反向环境变量缺口**：现行烛照九阴环境层只计「跌方向风险」，**看不见「国家队托底」这个反向力**——见 §八。

## 八 · 对 S2 计温 function 的启示（本 case 落剑酒青丘的意义）

### 8.1 已验证有效的部分
