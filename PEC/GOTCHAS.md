### 四、元层面陷阱（G-10~G-12）

**所在文件**：[[陷阱-G-06-G-14]] §五

- **G-10 框架自我保护** — 提醒：框架解释力有时效性，需定期检验
- **G-11 分析单元层次混淆** — **规则修订（2026-05-09 用户裁定）**：
  - Layer 1 国家行为体 → 充分可分析
  - Layer 2 特定组织（AIPAC 等） → 充分可分析
  - Layer 3 具体个人/机构 → 按标准分析
  - **Layer 4 族群/宗教/文明整体 → 允许输出，但必须 `[Layer 4]` 显式标注 + 列 Layer 1/2/3 依据 + 可证伪条件 + 内部异质性**

### 五、制度评估陷阱（G-13~G-14）—— "形式→实质陷阱三联"

**所在文件**：[[陷阱-G-06-G-14]] §五（G-13/G-14 与 G-12 并列）

- **G-13 部分纠偏误为完整纠偏**（2026-05-12 立案） — 强制：七维度分别评估 + 主动暴露弱维度
- **G-14 程序完备误为实质成功**（2026-05-12 立案） — 强制：计算实质比例（规模/边际/机会成本）+ 比较实际后果

**三联关系**：
- G-12 = 精密度诱惑（术层→本层错误）
- G-13 = 部分代表整体（维度层级）
- G-14 = 程序代表实质（形式→实质层级）

> **Claude 训练数据特别警戒**（CS-08 §六 方法论警戒第 11 项）：英语训练数据对西方制度的程序完备有结构性加权——环保主义叙事 / 多方参与叙事 / 审慎审查叙事被系统性合理化。对西方制度案例须**强制引入比例性诊断**。

> **G-20 一句话**：跨时段/跨文明的**结构相似 ≠ 传承**；传承须有**带指纹的事实传递**（被引文本/被搬形制/被教正典/制度连续），证明不了的降级为趋同/功能/待证。文化传承不要求世系（日本承唐宋是横向事实传承），但要求可证传递。同族 G-01/G-08/G-06，是「无默认值原则」在传承主张上的应用。
>
> **G-19 一句话**：**真诚 ≠ 非工具**；二者正交可并存，一个信念可被真诚持有、同时承担支配-合法化功能。真诚常**反向增强**叙事合法化效力（最有用的包装由信徒亲手写，福山样本）。"他真诚"不豁免工具性分析，是「无默认值原则」在动机判断上的应用。同族 G-08，与命题7「真诚≠犬儒伪二分」咬合。
