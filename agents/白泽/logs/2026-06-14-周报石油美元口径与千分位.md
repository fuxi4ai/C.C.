# 会话日志 — 2026-06-14

**项目**：白泽大宗（周报自动化工作流）
**出场者**：白泽
**主题**：石油改国际美元口径 + 全表价格千分位

## 完成的工作

- 老师指令两条：周报石油用美元单位、所有价格加千分位分隔符 `,`。
- 探明现状：原油原取自 Tushare `SC.INE` 上海原油期货（553.5 元/桶，人民币 P0）；价格渲染处直接字符串拼接，无千分位。
- 验证「换国际美元油价源」可得性：实测 Tushare **无** Brent/WTI 外盘美元原油期货——`fut_daily` 仅国内六所（INE/SHF/DCE/GFE/ZCE/CFX）人民币品种，`index_global` 仅国际股指，`ft_daily` 接口不存在。如实回禀老师，首选方案不通。
- 老师二次定夺：改走 **web/LLM 补国际价**。WebSearch 实检索 Brent ≈ $86.5/桶（2026-06-12 周五收，当周 -4% 至 3 月低点，美伊和谈预期压制）；同比基线取 EIA 口径 2025-06 月均 $69.85 → YoY ≈ +23.8%（与原 SC 人民币口径 +23.6% 交叉印证，换算自洽）。
- 落代码（持久化，非仅本周）：
  - `scripts/reports/build_weekly_report.py`：新增 `_fmt_price()` 助手（整数去赘点 + 千分位），改 MD 表（第 250 行）+ HTML data（第 443 行）两处格式化；HTML 看板价格串复用此数据，JS 无需动。
  - `scripts/data_collection/fetch_futures_prices.py`：FUT_MAP 移除原油 SC（借 `keep` 机制自动留存原油 web 记录，下周不被人民币价覆盖）；同步文件头注释。
  - `data/commodity_prices_live.json`：原油记录改为 Brent 美元（price 86.5、unit 美元/桶、yoy 23.8、P2、confidence 60、asof 2026-06-12、口径注记齐备）。
- 重渲本周周报，验证通过：原油 `86.5 美元/桶 · +23.8% · P2 · 置信 60%` 进鲜价主表；千分位全表生效（碳酸锂 175,300 / 六氟化钨 1,120,000 / 钨 500,500 / 稀土 697,500 / 铜 104,660），价微者照常。

## 做出的决策

| 决策 | 原因 | 影响 |
|------|------|------|
| 石油改国际美元口径，走 web 补价层（非 Tushare 换源、非 SC 汇率换算） | 老师两次定夺：先选换源，验证 Tushare 无外盘美元原油后改选 web 补价 | 原油从 P0 期货降为 P2 web 补价；每周 Stage1 web 刷新 Brent 美元价 |
| 同比基线取 EIA 2025-06 月均 $69.85，YoY 锚算 +23.8% | 与原 SC 人民币口径 +23.6% 交叉印证，守数据真实铁律 | 同比口径=当前点价 vs 去年月均，已在 source 注明；环比留「待核验」 |
| 千分位用 `_fmt_price` 统一格式化，整数去赘点 | 既加 `,` 又不引入无意义小数（553.5→553.5，500500→500,500） | MD + HTML 看板两端一致，持久生效 |
| 原油移出 FUT_MAP 而非保留换算 | 老师否决 SC×汇率换算；移出后 `keep` 机制保留 web 记录 | fetch 每周不再覆盖原油为人民币价 |
