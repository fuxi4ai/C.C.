## 💡 教训与经验

1. **防循环核心**: 同一操作 ≤ 2 次 → 第 2 次失败换方案；连续 3 次相似输出 → 停手
2. **session 重建后必须刷新 cron sessionKey** — 05-13 首次发现，05-24 流程化，两次实战验证
3. **长 session（400+ 消息）需 ≥900s 超时**
4. **cron 脚本必须用绝对路径导入** — 相对路径在不同 cwd 下 ImportError
5. **Archive 定期清理效果显著** — 单次可减 68% 冗余
6. **tushare 日期格式 YYYYMMDD** — 需转 YYYY-MM-DD 才能匹配
7. **PRD 交付标准是硬规则** — checklist 全勾才算交付
8. **安静日系统自动运行良好** — cron 全部正常，无需担忧
