### 配置文件

**Gateway API 配置：**
```javascript
const GATEWAY_CONFIG = {
  apiUrl: 'http://localhost:8080/v1/chat/completions',
  model: 'gateway/jarvis',  // 哥哥的模型
  timeout: 30000,
  maxRetries: 3,
  authHeader: process.env.GATEWAY_API_KEY ? {
    'Authorization': `Bearer ${process.env.GATEWAY_API_KEY}`
  } : {},
};
```

---

### 环境变量（可选）

| 变量 | 说明 | 默认值 |
|------|------|--------|
| GATEWAY_API_URL | API 地址 | localhost:8080 |
| GATEWAY_MODEL | 模型名称 | gateway/jarvis |
| GATEWAY_API_KEY | API 认证密钥 | 无 |
