### 核心功能

**1. 单个视频识别**
```bash
node dva.js analyze <视频链接>
```

**2. 批量识别（开发中）**
```bash
node dva.js batch-recognize <链接文件.json>
```

**3. 查看待识别视频**
```bash
node dva.js list
```

**4. 查看统计信息**
```bash
node dva.js stats
```

```json

      "subtitle": "字幕内容...",
      "created_at": "2026-04-17T20:00:00+08:00",

    "total": 1,
    "recognized": 1,
```

---

### 核心流程

```
1. 获取视频链接
   ↓
2. 添加到数据库（status: pending）
   ↓
3. 连接浏览器（CDP 端口 9222）
   ↓
4. 发送消息给豆包
   ↓
5. 等待豆包回复
   ↓
6. 提取字幕内容
   ↓
7. 更新数据库（status: recognized）
   ↓
8. 每 10 条汇报一次"入库成功"
```

---

## 💡 关键技术点

**1. 串行发送**
- 1 条视频/次
- 避免豆包限流

**2. 进度汇报**
- 每 10 条汇报一次
- 便于监控和断点续传

**3. 及时入库**
- 识别完成立即保存
- 不依赖缓存

**4. 详细日志**
- 每步操作都有记录
- 便于排查问题
