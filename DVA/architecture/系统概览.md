## 定位

从抖音创作者视频中萃取投资智慧的标准化分析框架。

## 四模块架构

| 模块 | 功能 |
|------|------|
| `1-获得视频链接` | 爬取作者视频 URL（playwright）|
| `2-获得视频文本内容` | 字幕转写（DYD + sensevoice-v1 ASR，豆包链路已废弃）|
| `3-文本分析` | 8层智慧萃取：洞见/经验/反常识/方法论/警示/作者画像/知识图谱/思维地图 |
| `4-量化分析` | dvaq.js + TypeScript |

## 主命令

```bash
dva analyze-level1 "作者昵称"   # 全流水线
dva get-links <sec_uid>          # 只爬链接
dva search-author <作者名>       # 搜索作者 sec_uid
```

## 关键架构决策

- v6.0 输出按作者昵称隔离：`Reports/<昵称>/`，不同作者互不覆盖
- 字幕链路：DYD（douyin-downloader）+ DashScope 原生 ASR（非兼容模式）
- `dva-to-graphify.js`：有连接 graphify 的设计意图（DVA × 渊图 协同）
