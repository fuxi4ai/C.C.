## 二、最硬主线深挖：电子布 / CCL 涨价链（图谱真实结构）

```
特种电子布厂                     高速 CCL / 板材              下游应用
─────────────                  ──────────────             ────────
中国巨石 / 中材科技 / 宏和电子   ──supplies──▶ 利森诺克      ──▶ 高速 CCL ──used_in──▶ 英伟达 Rubin 平台
泰山玻纤 / 旭化成(日) / 日东纺(日)                (CCL/高速板材)                          英伟达 LPU
三井金属 / 三菱化学·瓦斯化学(日)                                     南亚新材 ──supplies──▶ 高速 CCL
                                                                   松下(日) ──▶ 马十材料(M10 CCL)
布种：二代布(低DK) / Q布(高频) / 7628 / E-glass薄布(1080/2116) / T-glass布
                                                    高速 CCL ──measured_by──▶ 台光高速CCL提价(2025)
载板支线：M-SAP 工艺 ──enables──▶ SLP 内载板 ──part_of──▶ 1.6T 光模块（深南电路 supplies SLP）
```

**逻辑要点**：
- **供给紧的真源在高端布良率**：Q布（石英）良率低、二代布中空纱不良——本批核心增量，也是涨价能持续的底层（非普通周期涨价，是高端产能爬不上来）。
- **传导链条**：布涨价 → CCL 成本抬升 + 提价能力（台光 2025 提价已坐实）→ AI PCB（Rubin/LPU）买单。越靠上游（布）弹性越大、越稀缺；CCL 是量价齐升的中枢。
- **A股卡位**：布端中国巨石/中材/宏和/菲利华（石英），CCL 端生益科技/生益电子(688183)/南亚新材/华正，PCB/载板深南电路/沪电/胜宏。
- **注意**：利森诺克（图谱标 CCL/高速板材枢纽）真身待核（渊图 GOTCHAS 有「利森诺克 Lisenoke ≠ 力森诺科=Resonac」甄别遗留）——引用前核实。

## 三、其余主线图谱结构深挖

### 3-A · AI 光互联 NPO/CPO + InP 衬底瓶颈（第 2 硬）

```
云者(磷化铟衬底) ──supplies──▶ 磷化铟(InP)衬底 ──used_in──▶ EML光芯片 ──part_of──▶ 800G光模块
                                    │                CW连续波光源 ──part_of──▶ 800G光模块
                                    ├─requires─▶ 金属铟          1.6T光模块 ──evolves_from──▶ 800G
                                    ├─◀constrains── 硅光方案      载体铜箔 ──used_in──▶ 800G/1.6T光模块
                              InP衬底出口管制 ──causes──▶ InP衬底供需缺口   NPO互联 ──evolves_from──▶ 超节点
```
**要点**：真正的硬瓶颈是 **InP 衬底双卡**——出口管制 + 上游金属铟稀缺，同时受硅光替代路线挤压。光模块升级 800G→1.6T→NPO/CPO 是需求侧拉动。**卡点稀缺性 > 模块厂**。A股卡位：华工科技（光引擎）、矽电股份、上诠科技（保偏光纤·待核上市）、云者/珠海鼎泰（InP 衬底·待核）。

### 3-B · 先进封装：CoWoS-L 国产算力枢纽 + CoPoS 演进

```
华为昇腾 / 盛合晶微 / 航天电器 ──supplies──▶ CoWoS-L封装 ──evolves_from──▶ CoWoS-S封装
昇腾950PR / 昇腾960 ──requires──▶ CoWoS-L         CoWoS-L ──requires──▶ ABF基板 / 直写光刻设备 / 500方形基板
台积电二代 CoPoS(玻璃中介层) ＝ CoWoS 面板级下一代演进（本批新增伞节点 concept_CoWoS）
```
**要点**：CoWoS-L 是国产 AI 算力（昇腾 950PR/960）的封装咽喉，供给方盛合晶微/航天电器/华为自有产线；上游卡 ABF 基板 + 直写光刻设备。玻璃基板/CoPoS 是绕开 CoWoS 产能瓶颈的下一代路线。A股卡位：航天电器、帝尔激光（玻璃/直写激光）、ABF 载板兴森科技/深南电路；盛合晶微未上市。

### 3-C · 国产 AI 芯片 ASIC：出口管制→国产刚需，卡产能

```
英伟达GPU出口政策(1:4国产配售) ──causes──▶ 昇腾910/950系列 ──supplies──▶ 字节跳动(AI基础设施)
中芯南方 ──supplies──▶ 昇腾910/950     中芯南方 ──constrains──▶ 华为昇腾（产能瓶颈·既供又卡）
双芯封装 ──used_in──▶ 昇腾950PR         字节自研推理芯片 ──competes_with──▶ 华为昇腾
```
**要点**：逻辑闭环＝出口管制（1:4 配售）逼出国产替代刚需 → 昇腾承接 → 但**卡在中芯南方产能**（图谱里中芯南方对昇腾既 supplies 又 constrains，是最硬约束）+ 双芯封装。云厂（字节）一边买昇腾一边自研 ASIC 对冲。A股卡位：海光信息、中芯国际（中芯南方母体·产能瓶颈直接受益）；昇腾链多非独立上市。**弹性最大但需甄别流片/量产兑现**。

## 六、一句话结论

**最硬＝电子布/CCL**（供给紧真源在高端布良率、非周期涨价，CCL 量价齐升中枢）；**第 2 硬＝光互联 InP 衬底**（出口管制+缺铟双卡的稀缺点）；**弹性最大＝国产 ASIC**（出口管制逼出刚需，但卡中芯南方产能、需甄别兑现）。三条的共同上游主线：AI 算力扩张 → 先进封装(CoWoS-L/CoPoS) + 高速互联(光模块/CCL) 的材料/产能瓶颈。
