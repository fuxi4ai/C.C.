# 渊图系统概览

## 核心脚本

| 脚本 | 用途 |
|------|------|
| `kg_ingest.py` | Obsidian Warehouse .md → 图谱 patch（每5篇滚动合并）|
| `kg_merge.py` | 合并 patch 到图谱 |
| `kg_rel_classify.py` | 边语义分类 |
| `wiki_autogen.py` | 生成/更新 wiki 卡片（度≥4，不加--force只补新增）|
| `kb.py` | 知识库语义检索 |
| `tushare_ingest.py` | A股财务/行情独立查询（不入图谱）|

## 关键规则

- Patch 命名：`mapping/_v2_{timestamp}_{N}篇.json`
- 高时效数据（股价、财务季报）**不入图谱**，用 `tushare_ingest.py` 独立查询
- v1 已于 2026-05-03 归档，v2 从空白重跑（v1 三大质量问题：节点类型全写 concept、边字段命名错误、关系类型超13种）
