# Brain Vault 备份策略

> 三层冗余，保证 brain/ 数据在任何单点失效时都能恢复。

## 三层冗余

| 层 | 介质 | 频率 | 恢复 RTO |
|----|------|------|---------|
| ① | **GitHub 远程仓库** | 每次 /save commit + push | 任何机器 `git clone` 即可 |
| ② | **本地 .git 历史** | 每次 commit | `git reset --hard` 时光机 |
| ③ | **iCloud Drive 镜像** | 每日自动 + 关键操作前手动 | 通过 macOS 时间机器恢复 |

## 实施

### 层①：GitHub（已就位）

仓库：`https://github.com/fuxi4ai/C.C..git`（私有）
分支：`main`
触发：每次 `/save` 命令自动 `git add -A && git commit -m "session: {主题}"`

**手动推送**（如果 /save 没推）：
```bash
cd ~/Documents/Claude/brain && git push
```

### 层②：本地 git（已就位）

`.git/` 目录内置版本历史。无需额外动作。

**恢复某次回退**：
```bash
cd ~/Documents/Claude/brain
git log --oneline           # 找目标 commit
git reset --hard <hash>     # 回到那个 commit
# 或保留当前改动只看历史：
git checkout <hash> -- <file>
```

### 层③：iCloud Drive 镜像

**方案 A · symlink 进 iCloud（推荐）**

把 brain/ 整体软链进 iCloud Drive，iCloud 自动同步：
```bash
cd ~/Library/Mobile\ Documents/com~apple~CloudDocs/
mkdir -p brain-mirror
ln -s ~/Documents/Claude/brain brain-mirror/brain
```
> ⚠️ iCloud 对软链的同步行为复杂，可能不可靠。优先用方案 B。

**方案 B · rsync 周期镜像（更可靠）**

跑 `~/Documents/Claude/brain/.tools/backup-icloud.sh`，每日 cron：
```bash
# crontab -e
0 3 * * *  bash ~/Documents/Claude/brain/.tools/backup-icloud.sh
```

## 恢复手册

### 全盘恢复（机器坏了）

```bash
mkdir -p ~/Documents/Claude
cd ~/Documents/Claude
git clone git@github.com:fuxi4ai/C.C..git brain
# 或：
git clone https://github.com/fuxi4ai/C.C..git brain
```

### 单文件误删

```bash
cd ~/Documents/Claude/brain
git log --all -- path/to/missing.md   # 找到被删的 commit
git checkout <hash>~1 -- path/to/missing.md
```

### iCloud 时光机

Finder → 进入 `~/Documents/Claude/brain/` → 顶部菜单"进入时间机器" → 选历史版本。

## 不进版本控制的内容

见 `.gitignore`：
- `.DS_Store`
- `.index/`（每次跑 build-backlinks 重新生成）
- `graphify/*/cache/`
- `.obsidian/workspace.json`（个性化 UI 状态）
- `*.swp`, `*.swo`（编辑器临时文件）

## 监控

每周一手动跑：
```bash
cd ~/Documents/Claude/brain
git log --since='1 week ago' --oneline | wc -l   # 一周 commit 数
git remote show origin                            # 远程同步状态
ls -la ~/Library/Mobile\ Documents/com~apple~CloudDocs/brain-mirror/  # iCloud 最后同步
```

健康指标：
- 每周 commit ≥ 3 → ✓
- `git status` 显示 clean → ✓
- iCloud 镜像 mtime 在 24h 内 → ✓
