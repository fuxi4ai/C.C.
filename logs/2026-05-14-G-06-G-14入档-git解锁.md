### git 长期解决方案
- 沙箱多次留下 HEAD.lock 残留 → 每次 commit 都报 cannot lock
- 加 git 全局 alias：`git unlock` 一键清当前 repo 所有 .lock 文件
- 同时 `setopt CSH_NULL_GLOB` 解决 zsh 空 glob 报错把整条 && 链断开的问题
- 验证：`git unlock && git commit && git push` 一次通
