# 会话日志 — 2026-06-06（策略二分法与公理型）

## 完成的工作

- **MACD 降为确认积木**：消融实证——恐慌盘含 MACD 确认 vs 去 MACD = **58%/+1.89% vs 51%/+0.5%**，MACD 作确认有效；抽到 `infrastructure/indicators.py` 共享，消掉重复。
- **公理/主动二分法**（借鉴数学，写进 `docs/策略spec`）：公理型(edge=结构必然，校准+逻辑审计) vs 主动型(edge=构造模式，walk-forward+OOS+成本)；准则「公理锚定上下文、主动扣扳机」。
- **strategies/ 按二分法重排**：公理型/ · 主动型/(威科夫) · 组合/(恐慌盘) · _research/，各带 README。
- **公理型「高位天量出货」实现**（`strategies/公理型/strategy_high_volume_distribution.py`）：减仓信号，信号后 5/10/20 日 −0.4~−0.8%（基准+0.3~+1%）、中位−3~−4.5%、6成跌；**量越天越偏空、下跌占比60%→66%单调** ✓。
- **「地量止跌」自证伪 → 证伪且反向**（`strategy_low_volume_bottom.py`）：消融 地量低位 vs 放量低位 = 差约 3%（地量低位 10日 −0.95% vs 放量低位 +2.39%）。

## 做出的决策 / 关键发现

| | 内容 |
|---|---|
| MACD 定位 | 单用≈掷硬币（无edge），作"上下文里的择时确认"有真价值 → 降为积木 |
| 二分法准则 | 公理型锚上下文、主动型扣扳机；纯主动单用大概率掷硬币 |
| 顶底不对称（实证） | 顶：天量=派发(看空)✓单调；底：地量≠止跌（反向），**底部需放量承接** |
| 量=换手符号随位置翻转 | 顶天量=派发、底放量=承接；"无量"在底=该发生的承接没发生→偏空 |
| 自证伪法 | 用消融让策略暴露真假（地量 vs 非地量），不假设成立 |
