## 做出的决策

| 决策 | 原因 | 影响 |
|------|------|------|
| 改名只做「概念/标签级」，不物理改名 | 物理改 `DVA-Database`/`3-文本分析` 会连带 `assertDvaDbPath` 硬编码 + launchd + Codex 多点爆破，性价比差；概念级拿 95% 好处近零风险 | 仅改 help/文档/UI 称呼，路径与 guard 零改动 |
| category 与 mode 正交（不绑死） | 「设计」类既有只下载也有下载+转写，需 mode 逐人可变 | category 管去向/消费者，mode 管深浅；金融⇒full、PEC语料⇒transcribe-only 固定，设计二选 |
| 打勾写回走 widget→回传→CC→Mac 命令 | Cowork artifact 只能调 MCP 连接器、碰不了本地 watchlist.json | 用 show_widget 交互表单 + 生成 set-category 命令，不让前端直写本地文件 |
| MVP 只做分类+路由+打勾，不建投递管线 | 控制范围；实际投递沿用现有产物目录 | VV/PEC/金融库对接留下一期 |
