## 关键认知沉淀

1. **v1.1 可复用性已证实**：
   - 北京外部验证（Doctor 用别人 prompt 出超越原作者结果）
   - 杭州内部验证（Doctor 用 CC 的 prompt 出最优结果）
   - 两次都不是单次幸运一击 → **v1.1 是稳定范式**

2. **3D 微雕的真实价值**：
   - 不是审美产物，是**政经知识可视化的可点击底图**
   - 这反推到所有工艺标准：地标边界清晰 / 信息分层 / 可热区切分 / 可绑文献
   - **审美只是表层，交互层才是本体**

3. **季节混搭技法**（v1.1 没明确写入，v1.2 候选）：
   - 冬雪 + 早春梅 + 薄暮 比单季更有时间感
   - 这是张岱（小众）+ 苏轼（主流）+ 时刻锚 的三层叠加

4. **文化锚分层**（v1.1 没明确写，v1.2 候选）：
   - 科学锚（GPS 坐标）
   - 文化锚（完整诗句 / 题款）
   - 历史锚（"始建元代 / 申遗 2024"档案条目）

5. **非视觉符号视觉化**（v1.1 §1.6 延伸）：
   - "鸽哨余韵" → 北京无视觉表达
   - "白鸽飞舞" → 杭州/江南可视觉化
   - 同一概念，不同城市可有不同视觉转化策略

6. **CC 工程纪律的薄弱环节**：
   - 自己写的警告，自己没回头扫
   - **必须建立"prompt 起草后强制 GOTCHAS 扫描"步骤**
