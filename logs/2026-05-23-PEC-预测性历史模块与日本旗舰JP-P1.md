- 明文化**切分**：采纳图尔钦/波普尔内核（结构推导 + 大胆 + 事前登记 + 硬证伪），挡江教授（江学勤/Predictive History）那套被项目 G-04/G-09/G-10 立过案的病灶——「**胆大在 content，不在 ledger**」。
- 写入**旗舰预测纪律**（结构推导 / 结构刚性 / 路径显性 / dated 可判 / 信息增量 / 明写证伪条件）。
- 写入**拉卡托斯式更正协议**：证伪后三步裁定——定层（术层 vs 本层 miss）→ 进步性更正须当场带新预测 / 退化性补丁记账预警 → 同框架连续 2 次退化补丁触发整体重估；对接 G-10。重估触发器从「命中率<40%」钝阈值**升级**为「退化累计」质性判据。
- **Doctor 纠正后大改**：撤回初稿的「反偏置双向打脸（α/α′ 强制对称下注）」+「surprising-if-true / 禁止默认值」+ 两条双向候选 FH-C1/FH-C2；改为**单向精选选题纪律**（只预测结构刚性 + 路径显性的易预测事件，不两头下注、不顾模型平衡）。江教授档表同步改（精选/单向，不再「宽覆盖/接受高错误率」）。
- **圈题日本**：新建 **JP 预测组**（挂 CS-08 态 F2 上下半身分裂），正式登记旗舰 **JP-P1**（登记在先，今日时间戳）。三腿合取到 2028-12：① 上半身续升（防卫费≥2%GDP续涨/反击能力/对华姿态不回调）② 修宪卡关（参院无 2/3 + 公投 gate，正式修宪未完成）③ 下半身不改善（能源自给<25%/粮食<42%/稀土镓锗对华不脱钩）。置信度 75%，近端检验点 2027 春党大会「修宪发议メド」。
- **web 核 baseline（2026-05-23）**：高市早苗任首相力推修宪、目标 2027 春设发议「メド」瞄第 9 条；防卫费 FY2025 已达 2%GDP/¥11T（提前两年）、FY2026 预算 ¥9T 创新高(+9.4%)。
- **web 核出 CS-08 校正**：修宪发议需衆参各 2/3——自民仅**衆院**单独 2/3，**参院加维新仍不过半、无 2/3**。CS-08 A03「2026-02 大选三分之二多数·修宪必要条件实质达成」对参院 gate 偏乐观（实证锚点，建议回写）。该缺口反成 JP-P1② 最硬支撑。
- **路径显性补充（Doctor）**：日本精英自报的「国家正常化 / 普通の国」公开路线（小沢《日本改造計画》1993 →中曽根「战后政治总决算」→安倍「战后体制脱却/积极和平主义」→北冈伸一，对应 CS-08 §三）。意义：① 可拿行为者自报里程碑当标尺，declared route 与实际进度落差直接可测；② 是 CS-08「分头论述闭环」活体——「declared 正常化路线 ⟂ 物质基础现实」= 上下半身分裂最可观测形态。

## 做出的决策

| 决策 | 原因 | 影响 |
|------|------|------|
| 模块形态 = 升级现有 register，非新建 SP-09 | 与 predictions-register 机制重叠，另起会触 G-12/G-09 | 同一台账加两器官 |
| 定档江教授档 | Doctor 选定（大胆下定论换信息量） | 后经 Doctor 收窄为「精选+单向」 |
| 撤回「反偏置双向打脸/α-α′ 强制对称」 | Doctor：反偏置该管「别让偏见扭曲判读」，不该变「组合必须两头对称」；野心过大 | 选题改单向精选；删 FH-C1/FH-C2 |
