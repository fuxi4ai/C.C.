## 做出的决策

| 决策 | 原因 | 影响 |
|------|------|------|
| 价格走独立层、不入 canonical 图谱 | 价格是时间序列（图谱节点只存一个当前值）+ 保「高时效不入图谱」铁律 | 新增 prices/ 层，白泽走 price-coupling-v1 契约 |
| commodity 名为主键、node_id 可选桥 | 多数大宗/材料图谱里无节点（钨/APT/保偏光纤/MLCC粉体） | 价格层独立可用，并暴露图谱缺失节点候选 |
| kg_ingest 一次过、price_extract 退役为回填工具 | 同一篇付两次 LLM 浪费；统一抽取器避免不一致 | 往后新文档单次双产出，回填走 price_extract |
| PDF forward-only（存量基线跳过） | Doctor 指示：PDF 只抽以后新入库的，不回填存量 124 篇 | price_index 基线标记，--batch 只进新 PDF |
| 去重键加 horizon | 现价/预测/历史对比同日同商品会被错误合并 | bug 修复；清层重建保证干净 |
| 清层重建而非补缺 | 价格层newborn可复现；避免两套抽取器近重复 | 丢 53 条 run 记录，由回填重生 |
