6. **地域非视觉文化符号**：鸽哨 / 桂雨 / 评弹 / 汽笛 / 秦腔 / 嗦粉声...
7. **新增第十章对标案例附录**：北京 + 杭州春景双拆解 + 我们 card03 差距分析 + **杭州 v2.0 重做 prompt 已成稿**

## 关键认知沉淀（已入方法论 v1.1）

1. **范式差距 ≠ prompt 水平差距**：我们与北京/杭州春景差距不是写 prompt 不熟练，是**方法论代际差**（视角 + 季节 + 视觉灵魂 + 后期投入四维都欠）
2. **3D 微缩 vs bas-relief 是 prompt 媒介词的细微差别 → 完全不同输出风格**：image-2 解读差异极大
3. **季节决定 70% 视觉气质**：选错季节后期再花再多功夫也救不回来
4. **后期投入是档次 50%**：AI 出图占完成度 50%，矢量后期占 50%；不做 = 业余，做了 = 档案级
5. **GPS 坐标作"档案锚"是高级感来源**：把海报变档案 · 极便宜的细节
6. **双语图例栏是信息密度的高级解**：主图保持简洁 + 信息全放图例栏
7. **中文 prompt 对中国地名 + 不直接触发英文 safety classifier 双优势**：以后中国城市 prompt 优先中文
