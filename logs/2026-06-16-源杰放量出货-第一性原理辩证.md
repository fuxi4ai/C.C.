# 会话日志 — 2026-06-16

**项目**：跨项目（投研方法论 / 个股案例）
**主题**：源杰科技（688498）6/12 放量出货 vs 6/16 +10% —— 「强手出货大概率跌」第一性原理的辩证

---

## 完成的工作

- 核实 6/12 盘口实样：放量大阴线 **-13.66%**，成交额 **87.86 亿**创上市以来历史天量，换手率 **19.62%**，主力净流出约 **9.93 亿**；6/11 融资余额 23.02 亿、占流通市值 **4.54%**（杠杆盘重）。
- 核实催化事实：美伊 MOU 真实——特朗普 6/14 宣布达成、伊朗 6/15 凌晨确认停战谅解备忘录、6/19 瑞士签字；内容含解除石油制裁、重开霍尔木兹、解海上封锁等（油价↓→通胀↓→Fed 加息尾部风险↓ 这条宏观骨架成立）。
- 核实源杰供应链落点：InP 全自主工艺，100G EML 已量产交付头部模块厂（用于 400G FR4 / 800G DR8）；2025H1 数通营收 1.05 亿、同比 **+1034%** 首超电信。高端 50G+ EML 国产化率不足 20%、仍海外主导 → 源杰是「国产替代窗口」高弹性标的，非确定龙头。
- 围绕「6/16 +10% 是不是『强手出货大概率跌』的反面案例」做了 7 轮第一性原理辩证，逐节收敛。

## 做出的决策

| 决策 | 原因 | 影响 |
|------|------|------|
| 「高位放量＝看空风险＝中期大概率跌」标准成立，CC 持此标准 | 6/12 资金/换手/阴线三者全对得上，看空风险不打折；Doctor 命题是「大概率」非「必跌」，概率表述本就稳健 | 作为基准判据保留 |
| 6/16 +10% **不是命题的反例（证伪），而是变量改变下的叠加** | 出货的本质是「制造下行压力」（一个力），价格是合力；MOU 催化引入第二股上行力短期压过下行力。反例应是「无催化却涨」，「有催化涨了」只是换了条件集 | 区分「反例 vs 叠加」「朴素规则层 vs 机制层」两个分辨率 |
| 6/16 严格说连「小概率」都不算 | base rate 是 ceteris paribus（无催化）陈述；有 MOU = 从「有催化分布」抽样，上涨在该条件下并不稀奇 = 变量改变，非尾部稀有事件 | Doctor 认同此细化 |
| 「认错回补」措辞作废 | 该词默认强手＝长线趋势机构；源杰中盘更可能游资短线驱动，急涨急跌不携带「信念反转」信息 = 换挡非认错 | 双方达成一致 |
| 强手须分层：机构（长线/信念驱动/出货后少回）vs 游资（短线/催化驱动/无记忆，踩踏换手反而利于反包） | 二者「回不回」机制不同，不能并成「都不回」；但无催化语境下游资也不回 → 与机构同向看跌 | 判断 6/12 性质需看龙虎榜席位定机构派发 or 游资踩踏 |

## 遗留问题 / 待办

## Doctor 校正（2026-06-16，记忆分拣时）

- **龙虎榜席位置信度调低**：算法交易时代席位极易人为操控，不作为判定机构/游资的主要依据 → 原「看龙虎榜定 6/12 性质」待办撤下，改以量能/相对强弱为主。
