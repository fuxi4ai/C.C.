| 接受 subtle 调色版为终版底图 | 雪意保留 + 浮雕厚度 + 红色强化 + 全部地标位置/字保留 | Phase 1+2 完整通关 |

## 关键工程经验沉淀（已入方法论 v1.0）

1. **APIYI 节流伪装为 `moderation_blocked / shell_api_error`** —— 5xx + 4xx 都该 retry，60s 冷却一次能区分真错与伪装错
2. **批量抽卡卡间 ≥ 30s** —— 否则同节点限流爆发
3. **image-2 edit 是轻 LUT 调色 ≠ 重画** —— 关键是 prompt 用 `subtle / very mild / slight` 程度词 + 显式 `PRESERVE` 保护区 + Camera Raw 数值锚（"like Blacks -10, Contrast +8"）
4. **程度词陷阱**：deeper / richer / aggressive 会被推到极端 → 必须 subtle / mild / slight
5. **风格凝聚力 > 细节精度**：抽卡选型时这一条覆盖一切
6. **AI 写字必错** —— 所有文字一律矢量后期上字，prompt 里预留空白
