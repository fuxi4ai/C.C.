## Doctor 关键修正(价值漏斗, 取代"重叠=最高价值")

- L0 需求侧信号(三谱重叠给): 仅证明"需求旺盛"+"跨地缘主要客户共同认可"。是注意力筛子非价值结论。

- L2 公司可投资价值(个股层): 叠加【业务纯度 + 研发投入】等→整体交龙鱼五力六维(尤其估值15/财务健康5/纯度=赛道营收占比)。

## partnership_depth schema (新原语, 加在 supplies/enables 边上)
四级:
- captive    自产自供(华为海思/达芬奇/CANN; Google对TPU自有IP部分)
- co-development  锁步共研·同步迭代·下代首发(原型: 博通↔Google七代TPU共同设计2014起; 盛合晶微↔华为昇腾3年锁定独家)
- strategic  重要非锁步(大客户但非共同定义roadmap)
- merchant   一锤子/通用商业供应
判据信号: 联合实验室 / roadmap同步节奏 / 单客户依赖度 / 下代首发或独家 / 股权纽带 / 长期协议(LTA)
存量缺口确认: 现有 supplies 边仅 weight(置信度), 无深度维度→必须补
