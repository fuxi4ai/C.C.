## 完成的工作

- **持仓录入**：holdings.json 首次记 shares+cost+现金；12 只（出腾讯，入泡泡玛特/炬光/阿里）；schema 扩 shares/cost/status；SCHEMA.md 同步。
- **持仓看板 v1→v5**（暖色·卡片页范式）：色=分组维度、光=财务健康分（六维·财务维，3.5金/≤3橙脉冲/≥4无光/无分灰）、条=成本vs利润占市值；三分组切换（分态/行业按环节/评分）；股价+市值（读公共行情库 market_data.db 只读）；港股折人民币（汇率 0.865）；字体统一思源宋体。
- **生成器化 + 数据/版式解耦**：`build_holdings_board.py` 生成器；`serve_board.py` 零依赖本地服务（GET 服看板 / POST /api/refresh 跑生成器 / /api/save-holdings 写盘带备份+守恒校验）；`启动看板.command` 双击起服务。
- **抽共享层 `board_data.py`**：读四源+组装 payload+save_holdings 单一真源逻辑，生成器/服务/MCP 三家共用。
- **MCP 桥闭环（Step1-4）**：`longyu_holdings_mcp.py`（4 工具 get_board_payload/save_holdings/set_hk_price/set_rate），注册进 claude_desktop_config.json → Desktop 桥进 Cowork VM → artifact 用 callMcpTool 实时读写本地龙鱼库；完整看板搬进 Cowork artifact（侧栏常驻、编辑存回、无需本地服务）。
- **现金三账户**：holdings.json._meta.cash_accounts（太平洋 33490.58 / 招商 460626.90 / 招商两融 1083601.22，合计 157.8万）；board_data 求和为总现金；现金 pill 点击弹二级编辑面板（两看板都加）。
- **MCP 热重载**：MCP server 每次调用前 importlib.reload(board_data)，以后改数据逻辑免重启。
