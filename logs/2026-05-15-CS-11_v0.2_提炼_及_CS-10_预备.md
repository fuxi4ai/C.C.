## CS-10 俄苏立案预备清单（待 Doctor 决定的关键问题）

### 〇、CS-10 的特殊性预期

CS-10 是项目里**唯一同时满足以下多个验证目标**的案例：

1. **H-08 态 E 第二样本**（与中国 CS-05 对照）—— 验证 H-08 + F-08 v2 跨案例稳定性
2. **a.8 三阶段轨迹完整验证**（沙俄帝国自称 → 苏联自称 → 后苏联否认 = 完整时序）—— v2.8 立案的 Level A 升级强测试
3. **L1.补充 第六条家长制三种形态**（沙皇 / 苏联 / 后苏联）—— 跨代际失败的实证锚点
4. **F-04 离岸平衡的反向样本**（俄罗斯长期抗拒西方化 vs CS-08 日本被锁定）
5. **欧洲对俄敌意的反向校验**（B2 浮现 · 俄罗斯如何看待欧洲？身份认同的爱恨交织）
6. **杜金"第三罗马"+ 欧亚主义 vs 西方启蒙的文明对立**（杜金已在项目学者谱系）

### 一、命名候选（待 Doctor 决定）

| 候选 | 优点 | 缺点 |
|------|------|------|
| **CS-10_俄罗斯-苏联文明** | 与 CS-02/CS-03 盎格鲁撒克逊命名同构 | 略长 |
| CS-10_俄苏文明 | 简洁 | 历史早期（基辅罗斯/沙俄）覆盖弱 |
| CS-10_俄罗斯文明 | 最简 | "苏联"阶段被吸收，但苏联独立性强 |
| CS-10_东斯拉夫-俄罗斯文明 | 含文化层级 | 太学术 |

**推荐**：`CS-10_俄罗斯-苏联文明`（与 CS-02/CS-03 命名同构 + 突出两阶段独立性）

### 二、时间跨度候选

总跨度约 **1150 年**（基辅罗斯 882 起算）：

| 阶段 | 时间 | 关键事件 |
|------|------|---------|
| 基辅罗斯 | 882-1240 | 留里克王朝 / 988 东正教化（瓦西里皈依）|
| 蒙古统治 | 1240-1480 | 钦察汗国 240 年（亚洲化欧洲他者印象的起源）|
| 莫斯科公国 | 1283-1547 | 1480 摆脱蒙古 / 伊凡三世 |
| 沙俄帝国期 | 1547-1721-1917 | 1547 伊凡雷帝称沙皇 → 莫斯科第三罗马 / 1613 罗曼诺夫 / 1721 彼得大帝建帝国 / 1812 拿破仑征俄 / 1861 农奴解放 |
| 苏联期 | 1917-1991 | 二月+十月革命 / 1922 苏联成立 / 1941-1945 卫国战争 / 1953 斯大林死 / 1991 解体 |
| 后苏联期 | 1991-2026 | 叶利钦休克疗法 / 1999 普京上台 / 2014 克里米亚 / 2022 俄乌战争 |

### 三、主框架候选

按 CS-11 经验，可选三种模式：

| 模式 | 内涵 | 学术增量 |
|------|------|--------|
| **A · 假设驱动型**（CS-11 同构）| 以 H-08 态 E 第二样本验证 + F-08 v2 跨案例验证 + a.8 三阶段轨迹为骨架 | 与 CS-11 形成方法论同构对照 |
| **B · 文明扫描型**（CS-01/02/03 同构）| F-07 单应用 + 7 工具完整扫描 | 与既有 5 个 F-07 案例形成横向对照 |
| **C · 混合型**（CS-08 同构）| F-07 单应用 + 三视角对照 | 加入俄罗斯内部/中华/西方三视角 |

**推荐**：**A + C 混合**（假设驱动 + 三视角对照）—— 既验证 H-08/F-08 v2 + a.8 三阶段，又用三视角对照避免单视角偏置。
