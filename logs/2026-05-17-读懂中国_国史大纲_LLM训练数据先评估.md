## 方法论沉淀

### 1. 「LLM 训练数据先评估」方法论（C-03 候选）

**评估流程**：
1. 列出研究涉及的经典 / 知名作品 / 公开历史
2. 五维评估（文本 / 注疏 / 现代学术 / 西方汉学 / 当代议题）
3. ★ 五档评分
4. 判断：全 ≥ ★★★★ → 直接问 · 不需补料 / 任一 ≤ ★★★ → 评估补料需求

**首次应用**：CS-05 读懂中国 22 部经典 · 全部高覆盖 · 直接问。

**项目意义**：与 SP-03 质疑者 + SP-08 政治传闻处理 + 立案触发阈值 v2 共构**项目工作流前置评估纪律**。

### 2. 文化层 reference 写作纪律

- ❌ 不重复书的内容（LLM 已知）
- ❌ 不入主案例 CS-XX 主框架
- ✓ 重点放在**与项目主框架的对话**
- ✓ 重点放在**作者的局限**（避免单边赞美）
- ✓ 重点放在**当代意义**（连接 Doctor 命题）

国史大纲专题严格遵守此纪律。

### 3. 跨时空精神呼应作为分析视角

钱穆 1940 vs Doctor 2026 同议题 86 年间隔 · 不只是巧合 · 是**中华文明长周期议题的持续性**。这种跨时空对照是 CS-05 文化扩展的独特价值。
